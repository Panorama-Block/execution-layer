# Ecosystem Strategy

## Why Chains Fund Projects

Blockchain ecosystems provide grants because they want:
- Real transaction volume on their chain
- Protocol integrations generating TVL
- Visible usage and activity metrics
- Developer infrastructure others can build on

## PanoramaBlock Ecosystem Value Proposition

PanoramaBlock generates measurable on-chain value through:
1. Execution contracts deployed on the chain
2. Real swap/LP/staking volume through chain protocols
3. User activity attributed to PanoramaBlock infrastructure
4. Multi-protocol integration driving ecosystem engagement
